02 / ABOUT THE PROJECT

Not an image.
A relationship.

Five layers of thinking behind a living instrument.

01
INSPIRATION

Art that notices you back.

Most digital art is something we only look at. We wanted to create an artwork that responds to the person experiencing it. Every interaction produces a different composition, so the work only exists while someone is using it.

02
WHAT IT DOES

Voice, motion, and memory become material.

Move through the field, switch input modes, shape palette, density and drift, randomize a seed, save fields, export stills, and record the artwork evolving. Microphone strength changes its energy; hand direction changes its sound.

03
HOW WE BUILT IT

A browser becomes a studio.

HTML Canvas and JavaScript render every contour live. The Web Audio API translates sound into energy, hand tracking maps gesture to the field, seeded systems create distinct worlds, and local storage preserves the traces you choose to keep.

04
CHALLENGE

Making algorithms feel intentional.

We limited the visual language to flowing contours, particles, glow, and one cursor-orb. Every visible control teaches the visitor how their behavior changes the piece, keeping complexity expressive rather than arbitrary.

05
WHAT WE LEARNED

Technology can be the medium itself.

The interaction, algorithm, sound, and visitor all contribute to the final artwork. ECHO / FIELD could not exist as a traditional painting or static image—it must be generated in real time from human presence.
